사용 안내 · 구현 가이드

이 페이지는 ① 예제 카탈로그를 로그인 없이 사용하는 방법과, ② 외부 인프라(메일·SMS·OAuth) 또는 배치 집계가 선행되어야 동작하는 모듈을 어떻게 구현하는지 설명합니다.

카탈로그 사용 안내

1로그인 불필요 — 카탈로그와 모든 예제 화면은 로그인 없이 바로 사용할 수 있습니다. 방문 시 데모 관리자(admin) 세션이 자동 적용되어, 권한이 필요한 관리화면(코드·메뉴·권한·배치 등)도 그대로 열립니다.
2실제 동작 — 각 카드를 클릭하면 인메모리 HSQL DB와 연동된 실제 화면으로 이동합니다. 목록·검색·페이징·상세가 모두 동작합니다.
3표준화 규칙 — 모든 테이블은 TB_ 접두 + 대문자 스네이크 + 감사컬럼(FRST_REGIST_PNTTM 등)으로 표준화했고, 매퍼는 7-DBMS(HSQL/PostgreSQL/MySQL/Oracle/Tibero/CUBRID/Altibase)로 제공됩니다.
4직접 로그인도 가능 — 상단의 로그인 화면에서 admin / 1 로 로그인하면 로그인 흐름 자체를 예제로 확인할 수 있습니다.

외부 인프라가 필요한 모듈 외부연동

메일 발송 (cop/ems)
외부 SMTP 메일 서버가 있어야 실제 발송이 가능합니다.
① spring-boot-starter-mail 의존 추가 → ② application.yml에 spring.mail.host/port/username/password 설정(사내 메일릴레이 또는 SMTP 사업자) → ③ JavaMailSender 빈을 주입받아 MimeMessage로 발송 → ④ 발송 이력 테이블에 결과 적재.
필요 조건: 운영 SMTP 계정·발신 도메인 SPF/DKIM 설정
SMS/MMS 발송 (cop/sms)
통신사/문자 사업자의 발송 게이트웨이 API가 필요합니다.
① 문자 사업자(예: NHN Cloud·알리고·AWS SNS) 가입 후 API 키·발신번호 사전등록 → ② REST 클라이언트로 발송 요청 → ③ 발송 결과 콜백(Webhook) 수신·저장.
필요 조건: 발신번호 사전등록(전기통신사업법), API 인증키
소셜 로그인 (ext/oauth)
구글·네이버·카카오 등 OAuth2 제공자 등록이 필요합니다.
① spring-boot-starter-oauth2-client 추가 → ② 각 제공자 콘솔에서 clientId·secret·redirect URI 발급 → ③ spring.security.oauth2.client.registration.* 설정 → ④ SecurityConfig에 oauth2Login() 활성화.
필요 조건: 각 제공자 개발자 등록, 콜백 URL 화이트리스트
캡차 / LDAP (ext/captcha · ext/ldapumt)
이미지 캡차 라이브러리(또는 reCAPTCHA), LDAP/AD 디렉터리 서버가 필요합니다.
캡차: kaptcha로 서버 이미지 생성하거나 구글 reCAPTCHA(사이트키·시크릿) 서버 검증. LDAP: spring-boot-starter-data-ldap + 디렉터리 접속정보 → 사용자 동기화 배치.

배치 집계가 선행되는 모듈 배치집계

사용현황 통계 (sts/bst·dst·rst·sst·ust)
통계 화면은 일배치가 미리 적재한 집계 summary 테이블을 조회하는 구조입니다.
① 원천 테이블(게시판 TB_BBS·접속로그 등)을 ② 일배치(이미 구현된 sym/bat Quartz 스케줄러)로 집계해 ③ TB_*_SUMMARY(발생일자별)에 적재 → ④ 통계 화면은 summary를 조회·차트 표시.
참고: 접속통계(sts/cst)는 라이브 쿼리 방식이라 이미 카탈로그에서 동작합니다. 나머지는 위 집계 잡 작성·스케줄 등록이 선행되어야 합니다.

다중 엔티티 연관이 큰 모듈 연관베이스

주소록(cop/adb) · 커뮤니티(cop/cmy) · 스크랩(cop/scp)
회원·조직·명함·게시판 등 다수 테이블과 매핑관계가 함께 얽혀 있는 다중 패널 화면입니다.
① 연관 테이블(회원·조직·명함·게시판 매핑)을 함께 표준화·이식 → ② 다중 패널(검색조건·목록·선택영역) UI 구성 → ③ 매핑 CRUD 연동. 단일 목록보다 범위가 크므로 연관 베이스부터 단계적으로 이식합니다.
